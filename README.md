# Portafolio de Proyectos de Ciencia de Datos

Este repositorio reúne una colección de notebooks desarrollados como parte de mi formación académica en ciencia de datos, así como proyectos personales enfocados en el análisis, transformación y visualización de datos.

Mi objetivo es mostrar cómo aplico conocimientos técnicos para resolver problemas reales usando Python, pandas, numpy, visualización con matplotlib/seaborn, y herramientas para procesamiento de datos a gran escala.

---

## Estructura del repositorio

- `/proyectos` → Proyectos aplicados, más desarrollados y completos.
- `/notebooks` → Ejercicios académicos reorganizados, limpios y enfocados en mostrar habilidades específicas.
- `/utils` → Funciones auxiliares y scripts reutilizables.

---

## Tecnologías utilizadas

- Python (3.10)
- Jupyter Notebooks
- pandas, numpy
- matplotlib, seaborn
- scikit-learn (para futuros modelos)
- Apache Spark (introducción, si aplica)
